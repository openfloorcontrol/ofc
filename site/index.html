Quick Start

terminal
# Install ofc
pip install ofc

# Initialize a new project
ofc init

# Add agents with personas
ofc add @data "skeptical data scientist"
ofc add @code "expert python coder"

# Let them collaborate
ofc run "analyze sales.csv and project next week"
